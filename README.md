# node-hue-daylights
simulate the progression of daylight with three Hue lights

config file lives at ~/.config/node-hue-daylights-config.json

sample config file:
    {
      "latitude"  : "39.14",
      "longitude" : "-84.5",
      "host"      : "192.168.11.3",
      "user"      : "cf3fb49f535ef1a8f95dc1423c357",
      "east"      : "1",
      "south"     : "2",
      "west"      : "3"
    }
